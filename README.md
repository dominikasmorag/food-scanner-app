## Opis projektu

Food Scanner to prosta aplikacja mobilna stworzona w React Native z wykorzystaniem Expo.  
Aplikacja umożliwia skanowanie kodów kreskowych produktów spożywczych oraz pobieranie informacji o produktach z publicznego API Open Food Facts.

Po zeskanowaniu produktu użytkownik może zobaczyć:
- nazwę produktu,
- markę,
- kategorię produktu,
- Nutri Score,
- wartości odżywcze.

---

## Wykorzystane technologie

- React Native
- Expo
- TypeScript
- expo-camera
- Open Food Facts API

---

## Funkcjonalności

- skanowanie kodów kreskowych,
- pobieranie danych z API,
- obsługa błędów,
- obsługa uprawnień do kamery,
- wyświetlanie wartości odżywczych produktu.
